GARY

Generative AI Reviewer for You – Breaking Barriers in Traditional Learning

Project Identity

Project Name: GARY (Generative AI Reviewer for You)

Team Members: Steve Akiyama, Maile Look, Varsha Narayan, Grant O’Connor

Project Timeline: Spring 2025 Capstone Project (In Progress)

Value Proposition

Problem: Traditional assignments often lack adaptability, leading to disengagement and limited understanding.

Target Audience: University students and instructors in large general education courses.

Core Features: Personalized feedback, AI-generated follow-up questions, automated evaluation, academic source integration via RAG.

Why it’s better: Provides dynamic, real-time feedback and adapts to individual learning needs.

🛠️ Technical Implementation

  • Frontend: React, HTML/CSS
  • Backend: Python, FastAPI, LangChain
  • Model: OpenAI GPT-4
  • Database: Qdrant (Vector Store)
  • Infra: Docker, AWS EC2
  • Collaboration: GitHub, JIRA
  • Unique Features: Retrieval-Augmented Generation (RAG)

Access & Usage

Repo: GitHub Repository

Instructions: Clone repo, install dependencies, and run FastAPI backend and React frontend. See README for full setup guide.

Documentation: Included in the repository with user guide and API reference.
